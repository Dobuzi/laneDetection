# Lane Detection w/ Front Camera

1. How to use

```
git clone https://github.com/Dobuzi/laneDetection.git
cd laneDetection
python3 -m venv venv
source venv/bin/activate
python3 -m pip install -r requirements.txt
python3 main.py
```
